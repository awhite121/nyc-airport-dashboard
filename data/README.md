# Data

This repo includes the cleaned modeling dataset and TLC taxi zone lookup.

## Included

- `data/processed/taxi_clean_for_modeling.csv` — cleaned Manhattan → JFK/LGA trips.
- `data/processed/taxi_clean_sample.csv` — smaller sample for quick testing.
- `data/reference/taxi_zone_lookup.csv` — taxi zone lookup table.

## Not included

- `data/raw/taxi_data.csv` — raw taxi extract is intentionally excluded because it is too large for normal GitHub usage.
